# Gym Workout Logging

Запись результатов тренировок пользователя в markdown-дневник.

## Файловая структура

```
~/shelf/gym/
├── training_plan.md          # Полный план тренировок (упражнения, схема, питание)
├── weekly-YYYY-MM-DD.md      # Недельный план с чеклистами
├── YYYY-MM-DD.md             # Дневник за конкретный день
```

## Формат дневника (YYYY-MM-DD.md)

### Заголовок
```
# Тренировка — 16.06.2026
# Тренировка ног — 16.06.2026  # если нужно уточнить группу
```

### Упражнение — таблица подходов
```
## 1. Приседания в Смите (ноги впереди, глубокие)
| Подход | Вес | Повторения | Тип |
|--------|-----|------------|-----|
| 1 | пустой гриф | 10 | Разминка |
| 2 | 25 кг | 6 | Рабочий |
| 3 | 25 кг | 6 | Рабочий |
| 4 | 25 кг | 6 | Рабочий |
| 5 | 25 кг | 6 | Рабочий |
| 6 | 15 кг | 10 | Добивка |
| 7 | 15 кг | 10 | Добивка |
```

### Прогрессия — внизу файла
```
- **Прогрессия:** след. тяжёлый день ног — приседания 30 кг (рабочие подходы), румынская тяга 75 кг (рабочие)
```

## Терминология пользователя

| Термин | Значение |
|--------|----------|
| разминка | Warm-up set, легкий вес на 8-12 повторений |
| рабочие | Working sets, целевой вес на 4-8 повторений |
| добивка | Drop set / back-off set, легче рабочих, 10-15 повторений |
| пустой гриф | Только гриф Смита (вес не учтён) |

## Правила записи

1. **В процессе тренировки** — пользователь диктует упражнение, вес, повторения, количество подходов. Записываю сразу в файл.
2. **Прогрессия** — если все рабочие подходы выполнены чисто (целевое количество повторений), на следующей тренировке +2.5-5 кг. Записываю внизу файла.
3. **Недельный план** — создаю в начале недели со списком тренировок. Помечаю выполненные `✅`.
4. **Вес грифа Смита не учтён** — пользователь указывает вес блинов без учёта грифа.
5. **Язык** — русский, термины как говорит пользователь (разминка/рабочие/добивка).

## Типовой сплит (из training_plan.md)

```
ПН — НОГИ / ЯГОДИЦЫ (тяжёлый)
ВТ — ВЕРХ А (жимы + плечи) → кардио
СР — КАРДИО (интервалы)
ЧТ — НОГИ / ЯГОДИЦЫ (лёгкий)
ПТ — ВЕРХ Б (спина + брусья) → кардио
СБ — отдых
ВС — отдых
```

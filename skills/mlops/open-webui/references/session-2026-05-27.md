## Commands Used

```bash
git clone --depth 1 --single-branch https://github.com/open-webui/open-webui.git
```
Full clone times out (1GB+ history through GFW).

### Python deps
```bash
cd ~/open-webui
uv venv --python 3.12
# Install deps first (requirements.txt, not pyproject.toml — avoids hatchling build)
uv pip install -r backend/requirements.txt
# Symlink the package instead of `uv pip install -e .`
cd .venv/lib/python3.12/site-packages/
ln -sf /Users/admin/open-webui/backend/open_webui .
```

### npm
```bash
# --ignore-scripts is critical: cypress and onnxruntime-node postinstall hang from China
npm install --ignore-scripts --registry=https://registry.npmmirror.com
# Then run only esbuild postinstall (required by vite)
node node_modules/esbuild/install.js
node node_modules/vite/node_modules/esbuild/install.js
```

### Build
```bash
./node_modules/.bin/vite build
# Output: ~/open-webui/build/ (223MB)
```

### Start
```bash
OFFLINE_MODE=true WEBUI_SECRET_KEY="test-key-123" \
  python -u -m uvicorn open_webui.main:app --host 0.0.0.0 --port 8080
```
`OFFLINE_MODE=true` prevents HF model download hang. Without it, server blocks at "Fetching 30 files" for sentence-transformers/all-MiniLM-L6-v2.

## Error Diagnostics

### HF model download fails
```
ERROR [open_webui.retrieval.utils] Cannot determine model snapshot path:
  huggingface_hub.errors.LocalEntryNotFoundError:
  Distant resource does not seem to be on huggingface.co
```
Root cause: SSL certificate validation fails (this Mac's cert store), compounded by GFW blocking HF. Even HF mirror fails on some files.

Fix: `OFFLINE_MODE=true` or download manually with `aria2c --check-certificate=false`.

### vite binary not found
Post `npm install --ignore-scripts`, `.bin/` directory was empty. Running esbuild postinstall scripts regenerated the symlinks.

### FRONTEND_BUILD_DIR mismatch
`env.py` expects `build/` at project root (`BASE_DIR / 'build'`). After `vite build`, output lands there. Make sure this directory exists before starting the server.

## Config Variables (from env.py + config.py)

| Variable | Default | Used for |
|---|---|---|
| `OFFLINE_MODE` | false | Skips all HF network calls, local_files_only for embedding model |
| `RAG_EMBEDDING_MODEL` | sentence-transformers/all-MiniLM-L6-v2 | Embedding model for RAG |
| `RAG_EMBEDDING_MODEL_AUTO_UPDATE` | true | Auto-download updates for embedding model |
| `WEBUI_SECRET_KEY` | auto-generated | JWT signing key |
| `WEBUI_AUTH` | true | Disable auth with false |
| `HF_ENDPOINT` | https://huggingface.co | Alternative HF endpoint (set to hf-mirror.com in China) |
| `REQUESTS_CA_BUNDLE` | system | SSL cert bundle (set to empty to skip SSL) |
| `FRONTEND_BUILD_DIR` | BASE_DIR / 'build' | Where built frontend is served from |

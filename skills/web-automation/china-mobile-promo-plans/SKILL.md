# China Mobile Promo Plans

## Overview
Поиск выгодных промо-тарифов China Mobile (移动) с большим объёмом национального трафика (全国通用流量) для использования в поездках между городами Китая.

## Ключевые понятия
- **全国通用流量** — национальный трафик, работает по всему Китаю (нужен для маршрутов между провинциями)
- **省内通用流量** / **本地流量** — только в провинции/городе, не работает в поездках
- **定向流量** — только для конкретных приложений (Douyin, iQiyi и т.д.)
- **合约期** — срок контракта (обычно 12-24 месяца)
- **首充** — первый платёж (обычно 50-100元 курьеру при получении)

## Источники для поиска

### 1. Сторонние агрегаторы (simkazhijia.com)
- **URL:** https://simkazhijia.com/yidongliuliangka/
- Показывает много тарифов с детальным описанием
- Некоторые тарифы устаревшие (2024 г.), но принцип тот же
- Есть кнопка "→→→点此申请办理" — ведёт в магазин оформления
- **Минус:** нет прямого чата с поддержкой на сайте

### 2. Официальный Tmall China Mobile
- URL магазина: https://zhongguoyidong.tmall.com/
- Раздел号卡: `/search.htm?search=y&category=50025119`
- **Проблема:** Tmall/Taobao не принимают программные клики и CDP-автоматизацию
- Нужна ручная авторизация через QR/SMS

### 3. JD.com (京东)
- Поиск: https://search.jd.com/Search?keyword=中国移动流量卡
- Тоже проблемы с программной автоматизацией

### 4. Baidu поиск
- Лучший способ найти актуальные тарифы
- Ключевые слова: `重庆移动 大流量套餐 促销 线上专享`, `移动 星语卡 全国通用流量`, `移动 星沪卡 185G`

## Проблемы и решения (из опыта)

### Авторизация на Taobao/Tmall
- Через browser_tools НЕ работает: CDP сессия теряется при переходах между доменами (login.taobao.com → tmall.com), программные клики/ввод не срабатывают из-за защиты Taobao
- **Обход:** открыть `https://login.taobao.com/qrcodeLogin.htm`, сделать скриншот QR-кода, сохранить в Downloads, пользователь сканирует с телефона. Или SMS-вход.
- SMS-код: ввести номер → нажать "获取短信校验码" → дождаться SMS → ввести код → нажать "确定"
- Если CDP-сессия потерялась: перезапустить Brave (`pkill -f Brave` → запустить с `--remote-debugging-port=9222`)
- Tmall/Taobao страницы не показывают результаты поиска через программный click/type — нужно использовать прямые URL

### JD.com (京东)
- При входе через WeChat/OAuth может требовать привязку аккаунта
- Безопасность JD блокирует веб-вход с подозрительных сессий — требуется вход через JD App
- **Вывод:** JD не подходит для программной автоматизации

### Чат поддержки 10086.cn
- Использует Vant UI (Vue.js) — программная установка value не срабатывает
- Кнопка отправки `.send-btn` имеет `display: none` пока поле пустое
- **Метод отправки сообщения:** установить value через `nativeInputValueSetter`, затем вызвать `Event('input')` с `bubbles:true`, затем показать кнопку `.send-btn.style.display = 'block'` и кликнуть
- Альтернатива: отправить через `Input.dispatchKeyEvent` (симуляция нажатия клавиш)

### Оформление на иностранный паспорт
- Большинство онлайн-промо тарифов требуют 中国身份证  
- Для иностранного паспорта (护照) нужно уточнять в поддержке магазина
  - На 3.kazhijia.cn: кнопка "客服帮助" ведёт на WeChat контакт
  - WeChat не предоставляет Accessibility API для чтения чата через AppleScript
  - **Лучший путь:** пользователь сам пишет в WeChat поддержку, спрашивает: "可以用外国护照（护照）办理激活吗？"
- Альтернатива: физический офис China Mobile (营业厅) — но там не показывают онлайн-промо тарифы

## Процедура

1. Узнать требования пользователя:
   - Город/провинция
   - Нужен национальный или локальный трафик
   - Бюджет в месяц
   - Есть ли китайское ID или только паспорт
   
2. Поиск через Baidu с ключевыми словами

3. Сравнение тарифов — фокус на **全国通用流量** (национальный)

4. Для оформления — дать пользователю прямую ссылку, т.к. browser_tools не работают с Tmall/Taobao

## Pitfalls
- Не путать 省内 (провинциальный) и 全国 (национальный) трафик —本地 не работает в поездках
- Промо-тарифы часто действуют только 12 месяцев, потом цена повышается
- Первый платёж (首充) 50-100元 делается курьеру при получении карты
- Некоторые тарифы без голосовой связи (无语音功能) — только интернет

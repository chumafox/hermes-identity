## How to Order

1. Go to simkazhijia.com, find the plan
2. Click plan name → detail page has "→→→点此申请办理" link
3. Clicking opens order form on 3.kazhijia.cn
4. Fill: name, ID number, delivery address, phone
5. Courier delivers, pay the top-up amount (50-100元) on the spot
6. Activate with courier's help or via China Mobile APP

## Foreign Passport Issue

These aggregator promos REQUIRE 身份证 (Chinese ID) for online ordering. If the user has a foreign passport:
- Ask in the 客服帮助 chat (opens WeChat QR)
- Try the order form anyway — some accept护照
- Best alternative: visit a China Mobile physical store with the listing URL

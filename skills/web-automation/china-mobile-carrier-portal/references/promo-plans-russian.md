# China Mobile Promo Plans — Russian Reference

Russian-language reference for finding discount China Mobile tariffs. See main SKILL.md for full Chinese-language procedures.

## Russian Key Concepts

- **全国通用流量** — национальный трафик, работает по всему Китаю
- **省内通用流量** / **本地流量** — только в провинции/городе
- **定向流量** — только для конкретных приложений (Douyin, iQiyi)
- **合约期** — срок контракта (12-24 месяца)
- **首充** — первый платёж (50-100元 курьеру)

## Foreign Passport

Most online promo SIMs require 身份证. For passport:
- Ask 客服 on the order page: "可以用外国护照（护照）办理激活吗？"
- Physical 营业厅 accepts passport but won't have promo tariffs
- Alternative: use China Mobile APP (some promos available in "选卡中心")

## CDP Automation Notes

Taobao/Tmall reject programmatic clicks/input. Only reliable approach:
1. Navigate to `https://login.taobao.com/qrcodeLogin.htm`
2. User scans QR code with phone → logs in
3. Then navigate to Tmall store with direct URL

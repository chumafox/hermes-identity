# Ironclad Links and Evidence Procedure

How to obtain "железобетонная ссылка" (ironclad link) on a China Mobile promo tariff — proof so solid the operator can't deny it.

## Goal

Get a URL that shows the tariff exists, with details (price, traffic, minutes), from an authoritative source the China Mobile营业厅 employee can't wave away.

## Source Hierarchy (Strongest to Weakest)

### ✅ Best: China Mobile Official Flagship Store on Tmall
- URL: `https://zhongguoyidong.tmall.com/`
- Category for SIM/tariffs: `/search.htm?search=y&category=50025119`
- Shows the store badge (官方旗舰店) — undeniable
- Has customer service chat (阿里旺旺)
- **Pitfall:** Requires Taobao login. CDP automation strongly detected.
- **Workaround:** User logs in manually, agent gives the direct URL.

### ✅ Good: China Mobile Official Flagship Store on JD
- URL: `https://mall.jd.com/index-1000001405.html` (中国移动官方旗舰店)
- Similar to Tmall
- JD account required (may flag as "security risk" if accessed from unusual network)
- **Pitfall:** "账号存在安全风险" — user may need to verify via JD APP on phone

### ✅ Acceptable: simkazhijia.com
- Shows plan details clearly
- Has working order links to 3.kazhijia.cn
- **Not official** — aggregator site, operator may dispute
- Best use: print the page, show to营业厅 staff and say "этот тариф есть, подключите мне такой же"

### ✅ Fallback: Baidu search results
- Baidu's AI summary ("百度AI") sometimes shows tariff details directly
- Not authoritative but useful for initial discovery
- Save the Baidu URL + relevant snippet text

## Procedure

1. **First check** simkazhijia.com for plan details and screenshot the plan card
2. **If the user has Taobao**: navigate to zhongguoyidong.tmall.com, search the plan name, take screenshot with store badge visible
3. **Save the URL** — the address bar URL itself is evidence
4. **Chat screenshot** — open 阿里旺旺 chat on the product page, ask "可以用外国护照吗", screenshot the answer

## What Makes a Link "Ironclad"

- Domain: `zhongguoyidong.tmall.com` or `mall.jd.com/index-1000001405.html`
- Shows store as "官方旗舰店"
- Shows price, traffic allowance, contract terms
- Has a working "立即购买" button
- Bonus: chat transcript where customer service confirms the plan exists

# Billing & Account Info Cheat Sheet

From 业务查询退订 page (shop.10086.cn/i/?act=service_query):

## Column Headers
| Column | Meaning |
|--------|---------|
| 业务名称 | Service/Plan name |
| 资费标准 | Price |
| 订购时间 | Subscription date |
| 生效时间 | Effective date |
| 失效时间 | Expiry / deactivation date |
| 退订时间 | Unsubscription date |

## Reading the Plan Switch

When a plan change is scheduled:
- Current plan shows 失效时间 = next month 1st (e.g., 2026-06-01)
- New plan shows 生效时间 = same date (2026-06-01)
- New plan's 失效时间 = "生效--" (no expiry yet)

## Billing Period

From 账单查询 page: **计费周期: MM月01日-MM月DD日**
- Standard China Mobile billing cycle: 1st to last day of month
- No "reset date" per se — billing period is calendar-month

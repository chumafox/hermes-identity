# Gym log format (~/shelf/gym/)

Файлы: `YYYY-MM-DD.md` (одна тренировка = один файл)

## Шаблон

```markdown
# Тренировка — YYYY.MM.DD

## <Упражнение> — <тип>

- <детали>
```

## Примеры

### Силовая (приседания в Смите)
```markdown
# Тренировка ног — 13.06.2026

## Приседания со штангой в Смите

| Подход | Вес | Повторения |
|--------|-----|------------|
| 1 | 20 кг (2×10 кг с каждой стороны) + гриф | 10 |
| 2 | 20 кг (2×10 кг с каждой стороны) + гриф | 10 |
| 3 | 20 кг (2×10 кг с каждой стороны) + гриф | 10 |
| 4 | 20 кг (2×10 кг с каждой стороны) + гриф | 10 |
| 5 | 20 кг (2×10 кг с каждой стороны) + гриф | 10 |

- **Всего подходов:** 5
- **Отдых между подходами:** 1 минута
- **Примечание:** вес блинов — 20 кг (по 10 кг с каждой стороны), вес грифа Смита не учтён
```

### Кардио (велотренажёр, интервальная)
```markdown
# Тренировка — 14.06.2026

## Велотренажёр — интервальная

- **Режим:** 20 сек на мощность → 1 мин отдых
- **Длительность:** 10 минут
- **Раундов:** ~7 (20+60=80 сек × 7 = 9:20, последний цикл неполный)
- **Примечание:** интервальная, мощность на 20-секундных отрезках
```

## Правила

- Дата в названии файла — по пекинскому времени (CST, UTC+8)
- Одно упражнение на файл (если несколько — отдельные файлы или секции)
- Вес грифа Смита не учитывается, если не указано иное
- Формат — plain markdown, без излишеств

# Shelf проекты (~/shelf/)

~/shelf/ — универсальная «полка» для двух типов содержимого:
1. GitHub-клоны экспериментальных инструментов (MisoTTS, open-webui)
2. Личные логи и дневники пользователя (gym-тренировки и т.п.)

Основные проекты теперь в ~/projects/, а не в shelf.

## Список

| Проект | Путь | Описание |
|--------|------|----------|
| open-webui | ~/shelf/open-webui | Self-hosted LLM web UI. GitHub: open-webui/open-webui |
| MisoTTS | ~/shelf/MisoTTS | Local TTS engine. GitHub: swivid/MisoTTS |
| gym | ~/shelf/gym/ | Дневник тренировок — markdown логи по датам (YYYY-MM-DD.md) |
| alipan | ~/shelf/alipan/ | AliyunDrive CLI — REST API wrapper + HTTP сервер + README |
| openhuman | ~/shelf/openhuman/ | AI assistant for communities — Rust + React + Tauri v2. GitHub: tinyhumansai/openhuman |

## Gym-логи

Формат и шаблоны: `references/gym-log-format.md` в этом скилле.

- Файлы: `~/shelf/gym/YYYY-MM-DD.md` — логи тренировок по датам
- План тренировок: `~/shelf/gym/training_plan.md` — полная программа питания + тренировок (5/нед, 16/8 IF)
- Дата по пекинскому времени (CST, UTC+8)
- Одна тренировка на файл
